# Derakht Backend

Backend API for a children's story / book platform built with Django.

## Tech Stack

- Python
- Django
- Django REST Framework
- PostgreSQL
- Docker / Docker Compose
- Nginx

## Main Features

- User authentication
- Blog module
- Shop module
- Story management
- Image handling
- Dockerized development setup

## Why this project matters

This project demonstrates my ability to structure a Django backend with multiple apps, containerize the service, and build API-driven features for a real product.

## Running Locally

```bash
git clone https://github.com/Arianhf/derakht_backend.git
cd derakht_backend
docker compose up --build
```
